# Tools

## Local Development
- `npm run dev` starts the Next.js app.
- `npm run build` verifies the production build.
- `npm run lint` runs ESLint.

## Database
- `npx prisma generate` regenerates the Prisma client.
- `npx prisma db push` applies the schema to local SQLite.

## Tests
- `npm test` runs Vitest unit and integration tests.
- `npm run test:e2e` runs Playwright smoke and PWA checks.
